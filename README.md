# Hunt-the-rabbit
a text-based RNG game

## Description
This is a text-based RNG game where users have to try and survive for as long as possible(note*this doesnt have a hp counter yet. coming soon on v2).

## Features
- RNG(random number generator)
- offline

## Usage
step 1: download the file
step 2: open bash on the windows subsystem for linux
step 3: type in python3 Hunt-the-rabbit
